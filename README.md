<h1> 概要 </h1>
ウエブアルバムから指定されたアルバム内の全写真の詳細情報を取得して表示する。 表示する情報は まだ未定（・メーカ(make) ・モデル(model) ・ISO ・絞り(fstop) 等を予定）

<h1> 環境 </h1>
Python 3.8.2/Flask 2.0.2/Werkzeug 2.0.3/Mysql/CGI/OpenCV

<h1> 目的 </h1>
ウェブアルバム上からアルバム内の写真の情報を取得出来たらいいなと思い作成しようと思いました。
