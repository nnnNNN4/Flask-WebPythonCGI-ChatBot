<h1> 概要 </h1>
ウエブアルバムから指定されたアルバム内の全写真の詳細情報を取得して表示する。 表示する情報は まだ未定（・メーカ(make) ・モデル(model) ・ISO ・絞り(fstop) 等を予定）<br>
tensorflowを使用したチャットボット（Q&A機能）
